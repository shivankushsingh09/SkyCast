SkyCast is a premium, real-time weather application built with HTML, Bootstrap 5, and JavaScript. It provides accurate weather data and forecasts using the [Open-Meteo API](https://open-meteo.com/).

## Features

-   **Real-time Weather**: Current temperature, humidity, wind speed, and more.
-   **7-Day Forecast**: Plan your week with accurate daily forecasts.
-   **City Search**: Search for any city worldwide with auto-complete suggestions.
-   **Geolocation**: Get weather for your current location with one click.
-   **Premium Design**: Glassmorphism UI, dynamic gradients, and smooth animations.
-   **Responsive**: Fully optimized for desktop, tablet, and mobile devices.

## Tech Stack

-   **HTML5**
-   **CSS3** (Custom + Bootstrap 5)
-   **JavaScript** (ES6+)
-   **API**: Open-Meteo (No key required)

## How to Run

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Or verify using a local server (e.g., `npx http-server`).
